# ivangeorgiev.github.io/gems
Ivan gems site





## Development Setup



```bash
$ py -3.8 -m venv .venv38
$ source .venv38/Scripts/activate
$ python -m pip install --upgrade pip
$ pip install -r requirements.txt
```

To run mkdocs live server:

```bash
$ mkdocs serve
```

Build

```bash
$ mkdocs build
```
