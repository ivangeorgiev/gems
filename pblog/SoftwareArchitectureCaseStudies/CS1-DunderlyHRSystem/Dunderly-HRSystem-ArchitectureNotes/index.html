Requirements

Functional Requirements

  1. Web Based
  2. Perform CRUD operations on employees
  3. Manage Salaries:
  4. Allow manager to ask for employee's salary change
  5. Allow HR to approve/reject request
  6. Manage vacation days
  7. Use external payment system

Non-Functional Requirements

  1. 10 concurrent users
  2. Manages 500 users
  3. Data volume forecast: 25.5GB
  4. Relational and unstructured
  5. Not mission critical
  6. Payment system file based interface
